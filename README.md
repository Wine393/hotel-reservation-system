# Hotel Lin Kar 404 — Hotel Reservation System 🏨
> C# Console-Based Hotel Reservation Application
> Mae Fah Luang University | Team: ERROR404

## 📌 About
A fully functional hotel reservation system built in C# supporting
a 10-floor, 50-room hotel with 5 room types and complete
reservation lifecycle management.

## ✨ Features
- 🛏️ View available rooms across 10 floors
- 📋 Make reservations with guest details & nights
- 🔍 Search rooms by type
- ❌ Cancel reservations
- 🧾 Auto-generated formatted receipt with check-in/out dates

## 🏠 Room Types
| Type | Price/Night |
|---|---|
| Single | $50 |
| Double | $80 |
| Twin | $90 |
| Deluxe | $120 |
| Family | $150 |

## 👤 My Contribution
